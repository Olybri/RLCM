Rayman® Legends Challenge Manager

RLCM is a small application which allows you to install and play in the secret training room in Rayman® Legends, as well as to change the rules of those challenges and play past challenges again.

Quick user guide

When launching RLCM for the first time, the game needs to be running so that the tool is able to locate the game directory automatically. The second step is to install the training room, yet this can only be done while the game is fully closed. You can easily uninstall and reinstall the training room whenever you want, as long as the game is not running.

To start using the features, go into any challenge of your choice, and click the "load challenge" button. RLCM will then seek for the current challenge informations and rules through the game's memory. Once done, you are able to edit those rules.

  • Seed : an hexadecimal value which determines how the level is generated.
  • Goal : the goal to reach in order to fulfill the challenge.
  • Limit : the minimal score one must get to achieve the challenge.

Once the modifications are applied, it is sometimes necessary to restart the challenge, especially if the seed has been changed. Note that you can load an online challenge as well, yet you won't be able to edit the rules.

To play again a past online challenge, you need to enter a challenge in the training room that matches with the past one, and edit the seed so that it is the same as the past challenge. A history of all the pas challenge's seed can be found here.

Frequently asked questions

Why can I not access Murphy's Dungeon challenge in the training room?

This painting in the training room points to a nonexistent level file, which causes the game just to ignore it. We've already tried to change the file name and the pointed level name, but that doesn't seem to work. If you find a workaround, please let us know!

Why does the tool says "seed not found"?

The first thing RLCM is looking for in the memory is the seed value of the challenge. Once found, it can easily reach every other informations needed. To find the seed, it looks for a specific pattern in the memory. However, it sometimes happens that the pattern is a bit different, and the tool can't find any matches. If so, try to restart the challenge and click the load button before starting the actual run. If it still doesn't work, try to exit and reenter the level again.

How can I help develop the project?

If you're interested in the project, wheter you're a programmer or not, you can contact us on the "contact" tab. Any kind of help is appreciated.

The project is open-source and available on github, feel free to fork and create pull requests.
